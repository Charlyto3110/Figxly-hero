# Figxly docs SPA

## Ejecutar local
Abre `docs/index.html` directamente en el navegador (usa hash router, no requiere servidor).

## Publicar en GitHub Pages
- Source: carpeta `docs/`.
- Home cargará en `/#/`.
- Assets usan rutas relativas (`./assets/...`) para funcionar en subpath como `/Figxly-hero/`.
